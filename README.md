# Stickforge Arena

A browser prototype for a stickman combat game focused on forging custom weapons, editing custom arenas, and fighting bots or a second local player.

## Run

Open [index.html](./index.html) in a browser.

## Features

- Forge weapons by changing frame, material, edge, length, balance, and temper.
- Save custom weapons to an armory and assign them to Player 1 or the rival slot.
- Paint a custom arena with walls, hazards, player spawn, bot spawns, and a rival spawn.
- Fight either a bot skirmish or a local keyboard duel.

## Controls

- `WASD`: move Player 1
- `F`: attack with Player 1
- `G`: dash with Player 1
- `IJKL`: move Player 2 in duel mode
- `O`: attack with Player 2
- `P`: dash with Player 2
